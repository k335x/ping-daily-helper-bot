# PingDailyHelperBot

Легкий Telegram-бот для особистих нагадувань: додає задачі, зберігає їх у Firestore і надсилає повідомлення у вказаний час (Kyiv TZ).

---

## Можливості

- **Додавання задачі та часу нагадування**
- **Введення дати й часу одним рядком** у зручних форматах
- **Планувальник**, що перевіряє майбутні нагадування та шле нотифікації
- **Перегляд списку задач** (з інлайн-кнопками *Готово* / *Видалити*)
- **Збереження у Firestore** (Firebase Admin SDK)

> Час інтерпретується у часовій зоні **Europe/Kyiv** та зберігається в базі як UTC.

---

## Вимоги

- Node.js **18+** (рекомендовано 20 LTS)
- Firebase проект + `serviceAccountKey.json`
- Telegram Bot Token
